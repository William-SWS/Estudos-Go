# Flow Control

Os exemplos deste diretório foram organizados como programas independentes, um por pasta, para evitar conflito entre vários `main()` no mesmo pacote.

## Como executar

Use `go run` apontando para o arquivo do exemplo:

```bash
go run "./go tour/flowcontrol/for/main.go"
go run "./go tour/flowcontrol/while/main.go"
go run "./go tour/flowcontrol/if_else/main.go"
go run "./go tour/flowcontrol/condicionais/main.go"
go run "./go tour/flowcontrol/condicionais_curta_curacao/main.go"
```
