# Estudos Go

Este repositório contém códigos de exercícios e práticas de lógica de programação em Go, baseados principalmente na documentação oficial da linguagem.

## 📚 Sobre

Repositório de estudos para aprendizado da linguagem Go, contendo exemplos práticos de conceitos fundamentais e exercícios de lógica de programação.

## 📂 Conteúdo

O repositório inclui exemplos práticos sobre:

- **Variáveis e Tipos**
  - `hello.go` - Programa básico Hello World
  - `variaveis_inicializador.go` - Declaração de variáveis com inicializadores
  - `declaracao_curta.go` - Declaração curta de variáveis
  - `inferencia_tipo.go` - Inferência de tipos
  - `valores_zero.go` - Valores zero das variáveis
  - `conversores_tipo.go` - Conversão entre tipos
  - `constantes.go` - Uso de constantes

- **Estruturas de Controle**
  - `for.go` - Laços de repetição com for
  - `while.go` - Simulação de while usando for
  - `if_else.go` - Condicionais if/else
  - `condicionais.go` - Estruturas condicionais
  - `condicionais_curta_curacao.go` - Declarações curtas em condicionais

- **Funções**
  - `funcoes.go` - Declaração e uso de funções
  - `funcoes2.go` - Exemplos adicionais de funções
  - `funcao_multipla.go` - Funções com múltiplos retornos
  - `valores_retorno.go` - Retorno de valores
  - `exercicio_funcao.go` - Exercícios práticos com funções

- **Pacotes e Bibliotecas**
  - `math.go` - Uso do pacote math

## 🚀 Como Executar

Certifique-se de ter o Go instalado em sua máquina. Para executar qualquer arquivo:

```bash
go run nome_do_arquivo.go
```

Exemplo:
```bash
go run hello.go
```
